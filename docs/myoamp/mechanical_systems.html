Mechanical Design

The mechanical system of MyoAmp translates electrical muscle signals into controlled, repeatable hand motion. The design prioritizes modularity, durability, and user comfort while enabling multiple functional grips.

Overview

The goal of the mechanical system of MyoAmp was to create the actual orthotic system to integrate electrical parts onto the users. This was done in a few parts: integrating boards and power electronics (bicep mount and battery mount), integrating driving system (forearm and wrist mount), and integrating onto hand (finger mounts). We wanted to create an orthotic system that could easily be fit to users dimensions and needs, as well as create a system that could successfully counteract cable drive with minimal interference and discomfort.

Key Mechanical Components

The following elements form the backbone of the mechanical system and directly influence performance, reliability, and ease of assembly.

Hand Frame

A 3D-printed skeletal structure that supports all moving components. The frame was optimized for weight reduction while maintaining stiffness under load.

Linear Actuators

Small linear actuators convert electrical input into linear motion. These actuators pull tendons to flex the fingers and release to allow extension.

Tendon System

High-strength string is routed through the fingers to mimic biological tendons. This approach simplifies actuation and enables smoother, more natural motion.

Grip Modes

The mechanical design supports three grip configurations, each optimized for common daily tasks. Grip selection is controlled electronically, while execution is handled entirely by the mechanical system.

  • Power Grip – for grasping larger objects
  • Pinch Grip – for precision tasks
  • Point Grip – for interaction and indication

Design Iteration & CAD

Multiple CAD iterations were created to refine finger articulation, actuator placement, and overall ergonomics. Rapid prototyping allowed the team to quickly test mechanical assumptions and incorporate feedback into subsequent designs.
